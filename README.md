# README

## Project

This project is an experimental chatbot that gradually learns important details about a user's professional background through natural conversation.

The goal is not to force users into a rigid form. The system should collect useful facts over time, organize them into a structured profile, and preserve that profile in a reliable way.

## Current Status

This repository is still in an exploratory stage. The architecture and implementation details are expected to evolve.

## Design Direction

The current direction is to:

1. keep conversations natural and flexible,
2. incrementally build a structured professional profile,
3. standardize important profile concepts across users over time,
4. avoid regenerating the full profile from scratch on every turn.

For the current architectural thinking, see `GUIDELINES.md`.
